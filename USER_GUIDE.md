# 🎉 LifeOS 现在有用户友好的界面了！

## 🚀 三种使用方式

### 1️⃣ 一键启动器（最简单）

```powershell
python start.py
```

会出现一个漂亮的菜单，让你选择：
- 💬 聊天模式
- 🌐 Web 服务  
- 🎬 功能演示
- 🔧 管理工具
- 📚 帮助文档

### 2️⃣ Web 界面（最漂亮）

```powershell
# 启动服务
python run.py

# 然后在浏览器打开
http://localhost:8000
```

你会看到一个漂亮的聊天界面：
- 🎨 现代化设计
- 💬 实时对话
- 📱 响应式布局（手机也能用）
- 🚀 快速开始按钮

### 3️⃣ 命令行聊天（最快速）

```powershell
python scripts\chat.py
```

直接在命令行中和 AI 对话

## 🎯 快速开始

### 第一次使用？

1. **加载示例数据**
   ```powershell
   python start.py
   # 选择 4 (管理工具)
   # 选择 5 (加载示例数据)
   ```

2. **体验功能**
   ```powershell
   python start.py
   # 选择 3 (功能演示)
   ```

3. **开始使用**
   ```powershell
   python start.py
   # 选择 1 (聊天模式) 或 2 (Web 服务)
   ```

### 日常使用？

**选项 A：Web 界面（推荐）**
```powershell
python run.py
# 打开浏览器访问 http://localhost:8000
```

**选项 B：命令行**
```powershell
python scripts\chat.py
```

**选项 C：启动菜单**
```powershell

```

## 🆘 常见问题

### Q: Web 界面打不开？
A: 确保服务在运行：
```powershell
python run.py
```
然后访问 http://localhost:8000

### Q: 显示"无法连接到后端"？
A: 检查服务是否启动，运行：
```powershell
python scripts\diagnose.py
```

### Q: 想换个界面？
A: 你可以：
- 修改 `web/index.html` 改变外观
- 创建自己的前端界面
- 使用 API 文档集成到其他应用

### Q: 可以远程访问吗？
A: 可以！修改 `.env` 文件：
```env
HOST="0.0.0.0"
```
然后其他设备访问：`http://你的IP:8000`
