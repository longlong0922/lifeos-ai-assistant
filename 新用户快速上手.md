# 🎉 新用户快速上手指南

欢迎使用 LifeOS AI 生活助手！这是一个 5 分钟快速入门教程。

---

## 📋 第一步：检查环境（1分钟）

### 1. 确认已安装 Python
打开命令行（PowerShell），输入：
```powershell
python --version
```
应该显示 Python 3.8 或更高版本。

### 2. 进入项目目录
```powershell
cd d:\code\progranm\lifeos-ai-assistant
```

---

## 🔑 第二步：配置 API（2分钟）

### 如果你有腾讯混元 API（推荐）

1. 打开 `.env` 文件（如果没有，复制 `.env.example` 并重命名为 `.env`）

2. 填入你的 API 密钥：
```env
# 腾讯混元配置（推荐）
HUNYUAN_SECRET_ID=你的SecretId
HUNYUAN_SECRET_KEY=你的SecretKey
HUNYUAN_MODEL=hunyuan-large

# 使用混元
LLM_PROVIDER=hunyuan
```

### 如果你有 OpenAI API

```env
# OpenAI 配置
OPENAI_API_KEY=你的API密钥
OPENAI_BASE_URL=https://api.openai.com/v1  # 或使用代理地址

# 使用 OpenAI
LLM_PROVIDER=openai
```

### 如果你暂时没有 API

```env
# 使用模拟模式（体验功能）
LLM_PROVIDER=mock
```

**注意**：模拟模式只是返回固定文本，无法真正对话。建议申请真实 API。

## 🎯 第四步：体验功能（2分钟）

### 方案 A：先加载示例数据（推荐）

1. 在启动菜单选择 `4`（管理工具）
2. 选择 `6`（加载示例数据）
3. 数据加载完成后，返回主菜单

### 方案 B：直接体验

1. 在启动菜单选择 `3`（功能演示）
2. 自动展示所有功能

### 方案 C：开始聊天

1. 在启动菜单选择 `1`（聊天模式）
2. 输入消息，比如：`你好`
3. AI 会回复并开始对话

## 💬 常用对话示例

### 习惯追踪
```
你：我今天完成了跑步！跑了5公里
AI：太棒了！跑步5公里可不容易。能跟我说说，今天是什么让你成功完成的吗？
```

### 做决策
```
你：周末要不要去爬山？我有点纠结
AI：我理解你的纠结。让我帮你理清思路。首先，你现在的身体状态怎么样？
```

### 设定目标
```
你：我想学好英语
AI：很好的目标！我可以帮你拆解成 21 天计划。你目前的英语水平如何？
```

### 每日简报
```
你：给我今天的简报
AI：[分析你的习惯和目标，生成个性化简报]
```

### 反思对话
```
你：今天感觉有点累
AI：是什么让你感到累呢？
```

**提示**：Web 界面支持手机访问！

## 📊 查看你的数据


- 你的目标和进度
- 你的反思记录
- 你的决策历史

## 🆘 遇到问题？

### 问题1：显示"连接失败"或"API错误"
**解决方案**：
1. 检查 `.env` 文件中的 API 密钥是否正确
2. 运行诊断：`python scripts\diagnose.py`
3. 测试连接：`python scripts\test_llm.py`

### 问题2：端口被占用（8000端口）
**解决方案**：
```powershell
# 查找占用端口的进程
netstat -ano | findstr :8000

# 关闭该进程（假设 PID 是 1234）
taskkill /F /PID 1234
```

### 问题3：数据库没有数据
**解决方案**：
```powershell
python scripts\load_sample_data.py
```
加载示例数据。

### 问题4：Python 版本太低
**解决方案**：
升级到 Python 3.8 或更高版本。

### 问题5：缺少依赖包
**解决方案**：
```powershell
pip install -r requirements.txt
```
