**适合：** 习惯用命令行的人、快速使用、开发调试

**启动方式：**
```powershell

# 选择 1
```

**适合：** 所有人、日常使用、功能完整

- ✅ 界面美观

## 💡 重要提示

### ⚠️ 关于 API 配置

- **没有 API？** 使用 Mock 模式（`LLM_PROVIDER=mock`）
  - 能体验所有功能
  - AI 回复是固定文本
  - 适合演示和测试

- **有 API？** 配置真实 LLM
  - 腾讯混元（推荐，国内快）
  - OpenAI（功能强大）
  - 查看 `.env.example` 了解配置方法

### ⚠️ 数据存储

- 所有数据存储在 `data/lifeos.db`
- SQLite 数据库，本地存储
- 可以随时备份或删除
- 完全掌控你的数据

### ⚠️ 端口占用

- Web 服务默认使用 8000 端口
- 如果被占用，需要先关闭占用的程序
- 或修改 `.env` 中的 `PORT` 配置
